A dynamic, grid-based retro Maze Game built entirely using Python's native `turtle` graphics library. This project utilizes programmatic matrix mapping to construct structural wall systems, processing real-time keyboard inputs to navigate a player avatar from a starting spawn coordinate to a designated escape point.

---

## 🚀 Quick Start

### 1. Prerequisites
You only need **Python 3.x** installed. The `turtle` module comes included with the Python Standard Library, meaning no third-party package dependencies (`pip` install) are required.

> **Note for Linux users:** If you experience a missing graphical backend or Tkinter module error, resolve it by running:
> ```bash
> sudo apt-get install python3-tk
> ```

### 2. Run the Game
Clone the repository and launch the maze runner application:

```bash
# Clone this repository
git clone [https://github.com/SN4KED3V/Maze-Game-Python.git](https://github.com/SN4KED3V/Maze-Game-Python.git)

# Navigate into the project folder
cd Maze-Game-Python

# Run the maze game
python maze_game.py
